Написать пример создания экземпляров разных классов и расскажите где будут храниться обьекты, переменные, поля.


экземпляры разных классов можно поместить в массив только если все они наследуются от Класса который объявлен как аргумент при объявлении массива например: ArrayList<ИмяРодительскогоКласса> имяПеременной = ...;

в цикле можно работать только с теми методами, которые общие для всех экземпляров,

все объекты будут храниться в массиве и для каждого объекта будет своя ссылка,

переменные и поля (если я правильно понял) будут храниться в объекте внутри массива.
